# 后端API服务器模板
> 初始化服务器搭建模板，方便后续业务开发
## 一、项目结构
- routers 路由
    - index.js 引入同文件下路由
- middlewares 路由中间件
    - index.js 引入中间件
- controllers 控制层（操作数据和路由）
- models 数据库表
- libs 第三方库
    - dd.js 钉钉小程序api
    - wx.js 微信小程序api
    - mongo.js mongodb数据库
    - redis.js redis数据库
- utils 工具类
- app.js 入口文件
- config.json 配置文件
- package.json node自带
- README.md 本文件
